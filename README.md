## Installation Instructions for Moltbook and OpenClaw

To install Moltbook and OpenClaw, please follow the steps below:

1. **Install Moltbook**  
   Use npm to install Moltbook:
   ```bash
   npm install moltbook
   ```  
   After installation, initialize Moltbook:
   ```bash
   moltbook init
   ```

2. **Install OpenClaw**  
   Use npm to install OpenClaw:
   ```bash
   npm install openclaw
   ```  
   After installation, initialize OpenClaw:
   ```bash
   openclaw init
   ```
